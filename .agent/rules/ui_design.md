# Rule: UI/UX Design System for Premium Music Web

## Objective
Create a visual experience that is "stunning at first glance," premium, and immersive.

## Design Tokens

### Color Palette
- **Background**: `#0a0a0c` (Deep Obsidian)
- **Surface**: `rgba(255, 255, 255, 0.05)` (Translucent Glass)
- **Primary Accent**: `#a855f7` (Vibrant Purple)
- **Secondary Accent**: `#3b82f6` (Electric Blue)
- **Text Primary**: `#ffffff`
- **Text Secondary**: `#94a3b8`

### Effects
- **Glassmorphism**: `backdrop-filter: blur(16px) saturate(180%);`
- **Border**: `1px solid rgba(255, 255, 255, 0.1)`
- **Shadows**: `0 8px 32px 0 rgba(0, 0, 0, 0.8)`

### Typography
- **Primary Font**: 'Outfit', sans-serif (Google Fonts)
- **Fallback**: System sans-serif

## Interaction Guidelines
- **Hover States**: Sublte scale up (1.02) and increased brightness.
- **Micro-animations**: Progress bars should have smooth transitions.
- **Immersive**: Background should have a subtle gradient or animated mesh.

## Prohibitions
- No plain colors (red, blue, green).
- No default browser scrollbars (style them to match the theme).
- No harsh borders.
