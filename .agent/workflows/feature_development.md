# Workflow: Feature Development

## Step 1: Design Phase
- Define the UI component using CSS tokens from `rules/ui_design.md`.
- Create a mockup/wireframe if needed.
- Select/Generate assets.

## Step 2: Implementation (HTML)
- Scaffold the component in `index.html` or a separate component file.
- Use semantic tags.

## Step 3: Implementation (CSS)
- Add styles to `styles.css`.
- Focus on layout first, then "Premium" details (blur, gradients).

## Step 4: Logic (JS)
- Add event listeners and state updates in `app.js`.
- Ensure modularity.

## Step 5: Validation
- Verify in browser.
- Check responsiveness.
- Polish animations.
